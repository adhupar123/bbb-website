Exploratory Data Analysis

Our analysis began with exploring the distribution and relationships of molecular descriptors in the dataset. These visualizations help us understand the characteristics of drugs that can cross the blood-brain barrier.

Example of raw data formatted into a table.

Correlation Analysis

Understanding how molecular descriptors relate to LogBB values and to each other is crucial for building an effective predictive model. The correlation analysis reveals which features have the strongest influence on BBB permeability.

Correlation analysis showing relationships between LogBB and molecular descriptors.

Critical Findings

TPSA (Topological Polar Surface Area) shows the strongest correlation with LogBB at -0.53, indicating that higher polarity reduces BBB permeability. Lipophilicity (MolLogP) shows a positive correlation of 0.28, suggesting more lipophilic drugs cross the BBB more readily.

Linear Regression Analysis

We performed simple linear regression for each descriptor against LogBB to quantify individual relationships and assess which features provide the most predictive power.

Impact each molecular descriptor has on LogBB value. Each plot shows the regression line and RÂ² value indicating fit quality.

Regression Insights

Individual linear regressions show varying predictive power across descriptors. TPSA demonstrates the strongest linear relationship, while formal charge shows minimal correlation. These results inform our multiple regression model.

Multiple Regression Model Performance

By combining all molecular descriptors in a multiple linear regression model, we can better predict LogBB values. The predicted vs actual plot shows how well our model performs.

Multiple linear regression - Predicted vs actual LogBB values. Points closer to the diagonal line indicate better predictions.

Model Performance

The multiple regression model shows reasonable predictive ability with RÂ² = 0.33. Points clustering around the ideal fit line (red dashed) indicate accurate predictions, while outliers represent drugs with unique properties not fully captured by our descriptors.

Feature Importance Analysis

Understanding which molecular descriptors contribute most to BBB permeability predictions helps prioritize which chemical properties to focus on during drug design.

Coefficient Interpretation

Negative coefficients (TPSA, Molecular Weight, Aromatic Rings) decrease BBB permeability, while positive coefficients (Heavy Atoms, Formal Charge) increase it. The magnitude indicates the strength of each effect.
